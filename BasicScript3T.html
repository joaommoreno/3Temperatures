Your scientific question is:

Does adaptation to laboratory decrease plasticity?

The data set corresponds to populations of Drosophila subobscura that come from 3 different locations Ad (Adraga, Portugal), Mo (Montpellier, France), Gro (Groningen, Netherlands) that were introduced in the laboratory. They have spent 28 generations in new laboratory conditions. Each population was replicated 3 times after foundation, having 3 independent replicates evolving in the same conditions.

The TA population is the control population, already adapted to the laboratory.

Previous work has shown that populations in the first generation show high variation between the three populations at generation 4 and high plasticity at 13 and 23 degrees. The data corresponds to differences in mean number of eggs from each population to the control population. This allows to remove potential confounding effects due to assaying these populations in different time points.

To test if plasticity changed throughout adaptation to the same environment you have to:

  • Import the data

  • Do an ANOVA to test if there are differences between temperature across generations for each population

  • Create a plot per population

Testing differences between generations across temperatures

### Ad
modelAd<-lm(Fecundity~Generation*Temperature, data=ad)

summary(modelAd)
Anova(modelAd)

#### Gro

modelGro<-lm(Fecundity~Generation*Temperature, data=gro)

summary(modelGro)
Anova(modelGro)

#### Mo
modelMo<-lm(Fecundity~Generation*Temperature, data=mo)

summary(modelMo)
Anova(modelMo)
